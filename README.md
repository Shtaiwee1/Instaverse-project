# Instaverse-project

**Overview:**
A single page Full stack social media application to create,like,view,edit,share and delete posts.

**Technologies and languages:**
- HTML
- CSS
- JavaScript
- JSX
- ReactJS
- Redux
- Moment
- Redux Thunk
- ExpressJS
- MongoDB
- Mongoose
- Material UI
- React-Bootstrap
- Git , GitHub
- Bcrypt
- JSON Web Token
-	Axios
- AntD
- Heroku
- Netlify
